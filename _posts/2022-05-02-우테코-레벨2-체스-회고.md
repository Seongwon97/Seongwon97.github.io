# 들어가며
우테코의 방학이 끝나고..다시 레벨 2기간이 찾아왔다.

방학때 부족한 독서와 스프링 학습등 여러 공부를 하려고 했는데..생각보다 방학이 빨리 흘러간것 같다😭
나름대로 제주를 다녀온 4일을 제외하고는 레벨1 기간처럼 하루종일 공부를 했던 것 같은데..시간이 빨리 흘러가 내 거창한 목표를 모두 이루지는 못했던 것 같다..

그래도 방학 기간동안에도 열심히 공부하였고 목표도 이펙티브 자바 학습을 제외하고는 많이 이룬 것 같다. 이펙티브 자바는 여유가 된다면 레벨 2 출퇴근을 하며 읽어보고자 한다.

# 레벨 2 첫 미션 회고
레벨 2의 시작은 레벨 1 마지막 미션으로 진행하였던 체스 미션 코드를 Spark framework에서 Spring framework로 변경하는 것이었다.

미션은 페어와 내 코드중 더 괜찮다고 생각하는 코드를 선택한 후, 해당 코드를 통해 미션을 진행하는 것이었다.
내 이번 미션 페어 `슬로`와 나 모두 상대방의 코드를 사용해도 상관없다는 생각을 갖고 있었으나 `슬로`의 코드를 보던중 체스 플레이어 도메인을 만든 것이 인상깊어서 슬로의 코드로 미션 진행하는 것을 추진하였다.
페어로 진행하는 미션 수행은 Spark framework를 사용하던 프로젝트를 Spring framework로 변환만 하는 것이었기에 매우 금방 끝날줄 알았다.
하지만..생각과 다르게 상당히 많은 시간이 소모됐던 것 같다.

둘 다 스프링을 우테코 이전에 학습했음에도 불구하고 테스트 코드 작성, JDBC 등 처음 접해보는 내용들도 존재하였고 레벨1 기간동안 자바만을 사용하였기에 기억이 가물가물했던 것 같다.

그리고 나의 경우 이전까지 개인적으로 스프링 학습만하고, 무언가를 혼자서 만들어본적이 없기에 더욱 헤맷던 것 같다. 특히 페이지 이동과 같은 내용에서 말이다..
이번 미션을 진행하면서는 스프링의 기초 지식 학습을 하고 리뷰를 통해 여러 학습을 할 수 있던 것 같다.

# 과거 Spring 학습에 대한 후회 & 앞으로의 다짐
나는 우테코를 지원하기 전에 Spring을 반년간 인터넷 강의를 통해 공부하였었다. Spring의 MVC와 같은 스프링 기본 강의들은 패스트 캠퍼스 강의를 듣고,
JPA의 경우 패스트 캠퍼스, 인프런을 모두 들었던 것 같다. 하지만 이번 미션을 진행하며 느낀 것은..이전까지 내가 완전히 잘못된 방법으로 학습을 하고 과거 학습 내용들이 거의 도움이 되지 않는 다는 것이었다.
우테코를 들어오기 전, 독학으로 공부를 할 때는 동작원리, 비슷한 기능들의 차이점, 테스트 등 여러 학습 내용들에 대해 의문을 갖지 않고 각각 기술들에 대한 학습법만 공부했던 것 같다.
그러다보니 현재 머리속에 남아있는 것이 하나도 없는 것 같다.

이번 레벨2 체스미션을 시작하면서 과거 학습에 대한 후회?를 하고 이번 레벨 2기간에는 스프링의 사용법을 익히는 것 뿐만 아니라 공부한 내용은 완전히 이해하고 넘어가겠다는 다짐을 하였다.

레벨2 기간동안 열심히 학습하여 스프링 마스터를 하고자 한다🔥
