# IntelliJ 설치 Step
## 1. IntelliJ사이트에 접속

## 2. IntelliJ 다운로드
IntelliJ는 유료버전인 Ultimate버전과 Community버전이 존재한다.
각각 지원하는 기능들과 Ultimate버전의 비용을 확인해보고 본인에게 맞는 것을 다운로드 하면 된다.
저는 학생 License를 얻어 Ultimate버전을 다운로드 하겠습니다.
<img src="https://images.velog.io/images/seongwon97/post/c8fd5564-ff8f-4a6c-8eeb-8fab017db000/2.PNG">

## 3. IntelliJ 설치
- next버튼 클릭하기
  <img src="https://images.velog.io/images/seongwon97/post/ed506cdf-3064-46d2-9187-6b7e8a73bf21/3.PNG">
- 저장 경로 설정 후 next
  <img src="https://images.velog.io/images/seongwon97/post/6a4642a1-4e27-4943-a32c-fb2d01a15eb8/4.PNG">
- 체크 내용을 확인하여 체크후 next
  - Create Desktop Shortcut : 윈도우 OS 환경 체크 (64bit)
  - Update PATH variable(restart needed) : 윈도우 환경변수에 자동으로 추가 할 수 있도록 체크
  - Update context menu : 프로젝트로 폴더 열기
  - Create Association : 자바 사용
    <img src="https://images.velog.io/images/seongwon97/post/e0d38c15-8e2f-4e08-b30c-e5bae800ea70/5.PNG">
- 설정 마치고 install하기
  <img src="https://images.velog.io/images/seongwon97/post/bacc4af8-b403-40dc-82f9-c82b6362a163/7.PNG">
  <img src="https://images.velog.io/images/seongwon97/post/ff351b4e-aa63-4034-8883-2660a0cec217/8.PNG">
- 설치를 마치면 PC의 재시작을 해줘야합니다. reboot을 설정하고 finish.
  <img src="https://images.velog.io/images/seongwon97/post/c7c723aa-624a-4db1-80df-8ff9db80b2a4/9.PNG">
- 동의 하기 후 continue
  <img src="https://images.velog.io/images/seongwon97/post/abad0954-52e1-4412-82c3-30427271bbc2/10.PNG">
- 라이선스가 있는 계정으로 로그인해주기 대학생의 경우 [학생인증]("https://velog.io/@seongwon97/IntelliJ-%ED%95%99%EC%83%9D-%EC%9D%B8%EC%A6%9D-%ED%95%98%EB%8A%94-%EB%B0%A9%EB%B2%95")을 통해 라이선스를 취득할 수 있으며 Community버전을 사용하는 경우 인증이 필요 없습니다.
  <img src="https://images.velog.io/images/seongwon97/post/99f9cf5d-6065-446f-9d25-29b4c1fe76e8/11.PNG">
  <img src="https://images.velog.io/images/seongwon97/post/e368475a-2959-48ea-820f-b5c2dec5d0a7/12.PNG">
